How to experience CP/M-80 3.0?

By Legacy Installer
April 7, 2022

In the last post, we saw how to use WordStar 4.0 on YAZE. In this tutorial we are going to experience CP/M-80 3.0. You can download the emulator package from here:

  • CP/M-80 3.0 for Linux
  • CP/M-80 3.0 for Windows

Using CP/M-80 3.0

It is very easy to start using CP/M-80 3.0. On Linux, extract the above archive and right click on the cpm3.sh file and choose "Run as a Program" or run the shell file the way you run any other shell file. On Windows, extract the above archive and double click the Runme.bat file. The Virtual Machine will start.

You can enter the DIR command to see a list of files.

That's it! We used CP/M-80 3.0, an operating system launched in 1984 after 38 years (at the time of writing this tutorial).
